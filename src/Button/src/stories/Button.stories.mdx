## Usage

Install component

```
npm i @uikit/button

yarn add @uikit/button
```

Import component into your parent component.

```js
import Button from '@uikit/button';
export default {
  components: {
    Button,
    //or
    'base-button': Button,
  },
};
```

Then use it in your `template`

```html
<base-button
  theme="secondary"
  size="sm"
  :loading="isLoading"
  :disabled="isDisabled"
>
  My button
</base-button>
```
