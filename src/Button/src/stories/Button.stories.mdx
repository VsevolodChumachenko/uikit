## Usage

Install component

```
npm i @strahovkaru/button @strahovkaru/spinner

yarn add @strahovkaru/button @strahovkaru/spinner
```

> **NOTE**
>
> It works with only with global register `@strahovkaru/spinner` component
>
> ```js
> import Vue from 'vue';
> import Spinner from '@strahovkaru/spinner';
>
> Vue.component('spinner', Spinner);
> ```

---

Import component into your parent component.

```js
import Button from '@strahovkaru/button';
export default {
  components: {
    Button,
    //or
    'base-button': Button,
  },
};
```

or register it globally after `@strahovkaru/spinner`

```js
import Vue from 'vue';
import Spinner from '@strahovkaru/spinner';
import Button from '@strahovkaru/button';

Vue.component('spinner', Spinner);
Vue.component('base-button', Button);
```

---

Then use it in your `template`

```html
<base-button
  theme="secondary"
  size="sm"
  :loading="isLoading"
  :disabled="isDisabled"
>
  My button
</base-button>
```
