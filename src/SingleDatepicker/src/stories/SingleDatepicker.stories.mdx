## Usage

Install component

```
npm i @strahovkaru/input

yarn add @strahovkaru/input
```

> **NOTE**
>
> For validation you have to use `Vuelidate` https://vuelidate.js.org/

---

Import component into your parent component.

```js
import Input from '@strahovkaru/input';
export default {
  components: {
    Button,
    //or
    'base-input': Input,
  },
};
```

or register it globally

```js
import Vue from 'vue';
import Input from '@strahovkaru/input';

Vue.component('base-input', Input);
```

---

Then use it in your `template`

```html
<base-input
  v-model="$v.value.$model"
  :meta="$v.value"
  label="Label"
  error="Поле обязательно для заполнения"
/>
```
