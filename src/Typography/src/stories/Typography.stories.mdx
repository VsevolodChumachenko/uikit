## Usage

Install component

```
npm i @strahovkaru/typography

yarn add @strahovkaru/typography
```

---

Import component into your parent component.

```js
import Input from '@strahovkaru/typography';
export default {
  components: {
    Typography,
  },
};
```

or register it globally

```js
import Vue from 'vue';
import Input from '@strahovkaru/typography';

Vue.component('typography', Typography);
```

---

Then use it in your `template`

```html
<typography align="left" type="h1" tag-name="p">Какой-то текст</typography>
```
