## Usage

Install component

```
npm i @strahovkaru/steps

yarn add @strahovkaru/steps
```

---

Import component into your parent component.

```js
import Steps from '@strahovkaru/steps';
export default {
  components: {
    Steps,
  },
};
```

or register it globally

```js
import Vue from 'vue';
import Spinner from '@strahovkaru/steps';

Vue.component('steps', Steps);
```

---

Then use it in your `template`

```html
<steps :steps="steps" :current-step="2" align="center" />
```
