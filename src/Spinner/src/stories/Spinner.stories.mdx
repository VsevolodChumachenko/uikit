## Usage

Install component

```
npm i @uikit/spinner

yarn add @uikit/spinner
```

---

Import component into your parent component.

```js
import Spinner from '@uikit/button';
export default {
  components: {
    Spinner,
  },
};
```

or register it globally after `@uikit/spinner`

```js
import Vue from 'vue';
import Spinner from '@uikit/spinner';

Vue.component('spinner', Spinner);
```

---

Then use it in your `template`

```html
<spinner :inline="true" size="sm" />
```
